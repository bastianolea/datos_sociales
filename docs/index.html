Repositorio de datos sociales

Author

Bastián Olea Herrera

En esta página se reunen conjuntos de datos sociales listos para usarse en cualquier análisis, exploración, o proceso de aprendizaje.

Cada uno de los conjuntos se aloja en un repositorio de GitHub, acompañado del código en el lenguaje de progamación R que obtiene y prepara los datos para su uso, además de otros scripts de R que hacen uso del dato para generar otros productos, como gráficos, tablas, o aplicaciones Shiny.

Cada elemento de la tabla corresponde a un conjunto de datos, y en las columnas aparecen iconos que describen sus características. Los iconos se explican en la siguiente leyenda:

Todos los datos son públicos. Esto significa que cualquier persona puede utilizar el código y los datos de las aplicaciones para adentrarse en el mundo del análisis de datos y aprender a programar en R, el lenguaje de programación usado para procesar estos datos.

Si deseas aprender a analizar datos con R, en mi sitio web tengo tutoriales para principiantes y usuarios avanzados. Además, constantemente comparto consejos y aprendizajes. También puedes usar mi sitio web para contactarme con cualquier duda que tengas.

Si reutilizas estos datos, o estos datos te han servido, ¡no olvides compartir con los demás!
